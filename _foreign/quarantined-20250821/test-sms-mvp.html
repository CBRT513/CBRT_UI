Quick Test Instructions

This page helps you verify that SMS messaging with pick ticket delivery is ready for MVP testing.

Step 1: Load the Test Script

Open your browser console and navigate to your CBRT app, then run:

import('./src/utils/mvpSMSTest.js')

Step 2: Run MVP Test

This will check Twilio, warehouse users, and pick ticket generation:

runMVPTest()

Step 3: Fix Any Issues

If the test finds issues, use these commands to fix them:

Create a test warehouse user:

createTestWarehouseUser("John Doe", "5551234567")

Enable SMS for existing user:

enableSMSForWarehouseUser("USER_ID", "5551234567")

Step 4: Send Test SMS

Send a test release notification to verify SMS delivery:

sendTestReleaseNotification("5551234567")
