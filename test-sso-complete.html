ðŸ“± Adding More Apps

For any new app (Time Off Requests, etc.), just:

// 1. Link the package
npm link @barge2rail/firebase-auth

// 2. In your App.js:
import { AuthProvider, ProtectedRoute } from '@barge2rail/firebase-auth';

function App() {
  return (
    <AuthProvider>
      <ProtectedRoute appName="Your App Name">
        <YourAppContent />
      </ProtectedRoute>
    </AuthProvider>
  );
}

That's it! The app instantly works with your SSO system.
